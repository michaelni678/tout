**Tout** provides utilities that make working with [`proc-macro2`] easier.

# Setup

Add this to your `Cargo.toml`:

```toml
[dependencies]
proc-macro2 = "1"
tout = "0.3.0"
```

See the [documentation](https://docs.rs/tout) for more information.

[`proc-macro2`]: https://crates.io/crates/proc-macro2
